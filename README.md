# RCB 2017 Mega Auction Strategy Analysis using SQL

## Project Overview

This project analyzes IPL historical data (2013–2016 seasons) to help Royal Challengers Bangalore (RCB) build an optimal squad for the 2017 IPL Mega Auction.

Using SQL-based analytics, player performance, team weaknesses, venue records, batting strength, bowling effectiveness, age distribution, and all-rounder contributions were evaluated to generate data-driven auction recommendations.

The objective is to move beyond intuition and use historical IPL data to identify players who can improve RCB's chances of winning the IPL title.

---

## Business Problem

Despite possessing some of the strongest batting lineups in IPL history, RCB struggled to consistently win championships.

This project aims to answer:

- Which players should RCB retain?
- Which player profiles should RCB target in the 2017 auction?
- Is RCB overly dependent on batting?
- Which bowling resources can solve RCB's biggest weaknesses?
- How can data-driven decision making improve squad construction?

## Major Insights

### Batting Core Must Be Retained

Data confirms that:

- Virat Kohli
- AB de Villiers
- Chris Gayle

formed one of the strongest batting cores in IPL.

### RCB's Biggest Weakness Was Bowling

Analysis shows several elite wicket-taking bowlers outperforming RCB's bowling unit:

- Jasprit Bumrah
- Sunil Narine
- Dwayne Bravo
- Ravindra Jadeja
- Imran Tahir

### Valuable All-Rounders Identified

The strongest dual contributors include:

- Andre Russell
- Marcus Stoinis
- Shane Watson
- Ravindra Jadeja

### Venue Performance Concerns

RCB performed strongly at:

- M Chinnaswamy Stadium

But struggled at:

- Wankhede Stadium
- Rajiv Gandhi International Stadium
- MA Chidambaram Stadium

---

## Auction Recommendations

Based on historical IPL performance data:

### Retain
- Virat Kohli
- AB de Villiers

### Priority Targets
- Andre Russell
- Sunil Narine
- Jasprit Bumrah
- Dwayne Bravo
- Ravindra Jadeja

### Strategy

Instead of investing heavily in additional batting stars, RCB should prioritize:

- Death-over specialists
- Wicket-taking bowlers
- Impact all-rounders
- Players with strong away venue performance
